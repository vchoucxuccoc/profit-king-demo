## 🚀 快速开始

### 前置要求
- Node.js 18+
- Python 3.9+
- Git

### 安装运行
```bash
# 克隆项目
git clone https://github.com/your-username/profit-king.git
cd profit-king

# 安装依赖
npm install
cd backend && npm install
cd ../backtest_service && pip install -r requirements.txt

# 启动服务
npm run dev
```

### 访问地址
- **前端界面**: http://localhost:5174
- **API服务**: http://localhost:8000
- **回测服务**: http://localhost:8081
