## Project Highlights

Profit King is a revolutionary AI-driven quantitative trading platform that provides users with professional-grade quantitative trading solutions through **intelligent automatic tool selection** and **multi-AI analyst collaboration**. No technical background required - simply describe your trading philosophy, and AI will automatically build a complete analysis system.

### Core Features

- **AI Automatic Tool Selection** - Intelligently match the best analysis tool combinations based on strategy descriptions
- **Multi-AI Analyst Team** - Multi-dimensional professional analysis including technical, fundamental, risk, and more
- **Professional Backtesting Engine** - Quantitative backtesting verification based on Backtrader
- **Real Market Data** - Alpha Vantage + Alpaca dual data source support
- **Real-time AI Decisions** - Intelligent trading signals driven by DeepSeek large language models
